### tuples - immutable
`t = (0, 1, 'hello')`
#### tuple <-> list
```python
>>> t = ('cc', 'aa', 'dd', 'bb')
>>> t
('cc', 'aa', 'dd', 'bb')
>>> l = list(t)
>>> l
['cc', 'aa', 'dd', 'bb']
```

#### tuple -> dict
```python
>>> bob = dict(name='Bob', age=40.5, jobs=['dev', 'mgr'])
>>> bob
{'age': 40.5, 'jobs': ['dev', 'mgr'], 'name': 'Bob'}
```
